# INFIZAP Landing Page

Landing page estática da INFIZAP pronta para publicação no GitHub Pages.

## Estrutura

- `index.html`: entrada principal do site no repositório
- `infizap-lp-v1.html`: página principal da landing page
- `Logo/`: logo e favicon usados no projeto
- `Imagem Do Site Bugado/`: imagens de referência e diagnóstico

## Publicação no GitHub

1. Envie estes arquivos para o repositório `Infizap-Landing-Page`.
2. Ative o GitHub Pages usando a branch principal e a raiz do repositório.
3. Abra a URL publicada para conferir o carregamento do `index.html`.

## Observação

A página principal foi mantida no arquivo original para preservar a implementação atual. O `index.html` apenas redireciona para ela.
